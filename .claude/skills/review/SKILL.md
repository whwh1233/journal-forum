# 代码审查技能

对当前更改进行全面审查：

## 1. 架构检查
- 设计模式遵循情况
- 依赖管理
- 可扩展性考虑

## 2. 代码质量
- 可读性和可维护性
- 错误处理
- 日志实现

## 3. 安全漏洞
- 输入验证
- SQL注入风险
- XSS漏洞
- 认证/授权
- 敏感数据暴露

## 4. 性能
- 查询优化
- 内存泄漏
- 资源清理

## 5. 测试
- 测试覆盖率
- 边界情况
- 集成测试

提供具体的文件路径和可操作的建议。
