# 初始化项目文档

创建一个全面的CLAUDE.md文件来记录代码库：

## 1. 项目概览
- 目的和目标
- 技术栈
- 关键依赖

## 2. 架构
- 目录结构
- 模块组织
- 数据流

## 3. 开发设置
- 安装步骤
- 环境变量
- 构建流程

## 4. 常见任务
- 运行测试
- 开发服务器
- 生产构建

## 5. 重要注意事项
- 已知限制
- 性能考虑
- 安全关注

将文件保存到项目根目录：CLAUDE.md
